# Mr. Meeseeks Box

devcontainer with optional tool integrations.

## Quick start

1. Copy the env file:
   ```bash
   cp .env.example .env
   ```

2. Open in VS Code and "Reopen in Container".

## Claude Code

Claude Code is pre-installed. To use it, open a terminal in the container and run:

```bash
claude
```

Complete the OAuth flow in your browser to authenticate with your Pro subscription. 

Coworkers without a Claude subscription can simply ignore it.
